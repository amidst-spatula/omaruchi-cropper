# omaruchi-cropper

おまるちのキャラ選択画面のスクリーンショットから各プレイヤーのキャラクターを抽出・整列して1枚に合成するよ

## 使い方
1. スクリーンショットを複数枚選択してアップロード。
2. 自動でプレイヤー名が解析され、同じプレイヤーが縦一列に並ぶように整列・合成されます。
3. 位置がズレている場合や並べ替えが正しくない場合は「位置調整を表示」を押します。
    - **キャラ枠調整 (青)**: 切り抜き位置がズレている場合に選択し、上下にドラッグして調整。
    - **名前解析枠調整 (ピンク)**: 並べ替え（ソート）が正しくない場合に選択し、プレイヤー名の位置に合うよう上下にドラッグして調整。
4. 設定したオフセットは自動保存されます。
5. 生成された画像をタップして保存。

## 特徴
- **自動整列**: プレイヤー名の画像解析（NCC: 正規化相互相関）によるオートソート機能。
- **個別オフセット調整**: キャラ切り抜き位置と名前解析位置を個別に調整可能。
- **デバイス補正**: ノッチやステータスバーを自動検出し、切り抜き位置を補正。
- **設定保存**: 調整値はブラウザ（localStorage）に自動保存。
- **プライバシー**: サーバーへの画像送信なし（すべてブラウザのCanvas APIで完結）。
- **外部ライブラリ不要**: Pure HTML / CSS / JS。
