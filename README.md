# omaruchi-cropper

おまるちのキャラ選択画面のスクリーンショットから各プレイヤーのキャラクターを抽出・整列して1枚に合成するよ

## 使い方
1. スクリーンショットを複数枚選択してアップロード。
2. 自動でプレイヤー名が解析され、同じプレイヤーが縦一列に並ぶように整列・合成されます。
3. 位置がズレている場合は「位置調整を表示」を押し、プレビュー画像を上下にドラッグして微調整してください。
4. 生成された画像をタップして保存。

## 特徴
- **自動整列**: プレイヤー名の画像解析によるオートソート機能。
- **デバイス補正**: ノッチやステータスバーを自動検出し、切り抜き位置を補正。
- **設定保存**: 調整したY軸オフセットはブラウザ（localStorage）に自動保存。
- **プライバシー**: サーバーへの画像送信なし（すべてブラウザのCanvas APIで完結）。
- **外部ライブラリ不要**: Pure HTML / CSS / JS。
